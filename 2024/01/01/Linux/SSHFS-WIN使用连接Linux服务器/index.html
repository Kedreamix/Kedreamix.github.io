SSHFS-WIN使用连接Linux服务器

发表于2024-01-01|更新于2024-01-28|Linux
|字数总计:165|阅读时长:1分钟|阅读量:

Installation

Step 1
Install SSHFS-Win on your Windows computer.
Follow they installation instructions before continue.

  • Install the latest version of WinFsp.
  • Install the latest version of SSHFS-Win. Choose the x64 or x86 installer according to your computer’s architecture.（一般win电脑是x64）

在您的Windows计算机上安装SSHFS-Win。 在继续之前，请按照以下安装说明进行操作：

  • 安装最新版本的WinFsp。
  • 安装最新版本的SSHFS-Win。根据您计算机的架构选择x64或x86安装程序。（通常情况下，Windows电脑使用x64架构。）

Step 2
Once SSHFS-Win is installed, download the lastest setup from the releases section and install it.
安装桌面版本的SSHFS-win，用最新版本
Step 3
Add your connections and enjoy!
